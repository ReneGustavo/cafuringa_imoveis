Ajuda

Como cadastrar imóveis?

Para cadastrar imóveis, use o botão "Cadastrar imóvel", localizado no menu de administrador.

Como editar imóveis?

Para editar imóveis, use o botão "Listar", localizado no menu de administrador, e em seguida, após localizar o imóvel que deseja realizar a operação, clique em no botão com um ícone de um lapis.

Como cadatrar fotos no imóvel?

Para cadastrar fotos, use o botão "Listar", localizado no menu de administrador, e em seguida, após localizar o imóvel que deseja realizar a operação, clique em no botão com um ícone de uma câmera. Não há limite de fotos para serem cadastradas simultaneamente.

Como manipular as fotos dos imóveis

Para manipular as fotos dos imóveis, use o botão "Listar", localizado no menu de administrador, e em seguida, após localizar o imóvel que deseja realizar a operação, clique em no botão com um ícone de uma imagem. Assim, todas as imagens pertencentes àquela foto serão listadas. A partir daí, você poderá excluí-las ou marcar uma como a foto principal do imóvel.

O que é foto principal? Como escolher?

Foto principal é a foto que mostrará o imóvel nas listagens gerais. Para escolhe-la, use o botão "Listar", localizado no menu de administrador, e em seguida, após localizar o imóvel que deseja realizar a operação, clique em no botão com um ícone de uma imagem e clique no botão de confirmar na foto desejada.

Como excluir um imóvel?

Para excluir um imóvel, use o botão "Listar", localizado no menu de administrador, e em seguida, após localizar o imóvel que deseja realizar a operação, clique em no botão com um ícone de "X". Lembre-se: não há processo reverso para a exclusão de imóveis.

Como cadastrar usuários?

Para excluir um imóvel, use o botão "Cadastrar Usuário", localizado no menu de administrador.

Como gerenciar usuários? (admin)

Para gerenciar usuários, use o botão "Usuários", localizado no menu de administrador. A partir daí, é possível listar todos os usuários e excluí-los, se necessário.

OBS: Apenas o usuário 'admin' tem acesso a essa ferramenta.
